# UI Phase C — Interactive Exploration UI

## Objective
Phase C expands Phase 1 into richer interactive drill-down pages and exploration tools while preserving the Phase 1 decision-support focus.

## Highlights (high-level only)
- Rich target detail pages with observing windows and finder charts
- Satellite detail pages with pass history and context
- Enhanced Sky Tonight page with contextual highlights
- Improved filtering and sorting of recommended targets

## Constraints
- Phase C assumes Phase 1 and Phase B foundations are stable
- Continue to favor mock-first development during rollout
